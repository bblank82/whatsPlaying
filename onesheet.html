What's Playing?

Home theater now-playing monitor & remote control

A whole-home dashboard that shows what's playing on every Apple TV and Kaleidescape player in real time — album art, title, progress, transport controls, and remote — all in a single browser tab, with no polling lag thanks to WebSocket push updates.

Supported Devices

Apple TV 4K (all generations)
Apple TV HD
Kaleidescape Strato / Terra

Core Features

🎬

Live Now-Playing Cards

Per-device cards display title, artist/album, series & episode, progress bar with elapsed/remaining time, and a blurred album-art background that updates instantly via push subscription.

🖼

Poster Art from TMDB

Movie and TV show posters fetched automatically from The Movie Database. For TV shows, season-specific art is resolved — even when the device doesn't report a season number — by matching the episode title against all seasons in parallel.

🎮

Transport Controls

Play/pause, skip forward/back, and scrub to any position directly from the card. Kaleidescape shows variable-speed scan and chapter navigation instead of fixed-seek buttons.

📱

Full Remote Control

Tap the remote icon to open a Siri Remote–style overlay: D-pad, select, menu, back, home, play/pause, and title/chapter navigation — all without leaving the browser.

⚡

WebSocket Push Updates

The backend maintains persistent connections to every device. State changes arrive in the browser within the poll interval (default 5 s) with no manual refresh required.

🔍

Auto-Discovery

Devices are found automatically via mDNS on the local network. Apple TVs on other subnets can be added by IP; they connect via AirPlay even when mDNS can't reach them.

🔌

Persistent Connections

Devices that go offline are retained and reconnected automatically on the next scan cycle. Known devices load immediately on startup so the UI is never blank.

📌

Device Pinning

Pin a device's IP so it reconnects directly by TCP even if mDNS goes silent — useful for Apple TVs across VLANs or when Bonjour proxy coverage is incomplete.

Kiosk Mode

Turn any browser tab into a dedicated now-playing display. Bind it to a specific device, a room (auto-switches to whatever is active), or the whole network. Supports both landscape and portrait orientations (90° CSS rotation) — ideal for a wall-mounted tablet, a rack-mount touchscreen, or a Raspberry Pi display. When nothing is playing the screen goes dark automatically. Configurable per connected client from the Settings panel.

Settings & Management

Device Management

  • Add any Apple TV or Kaleidescape by IP address
  • Assign devices to named rooms (Theater, Living Room, etc.)
  • Pin / unpin devices for reliable cross-subnet reconnection
  • Hide devices from the dashboard without removing them
  • Forget a device entirely, removing credentials and config

Pairing

  • Pair AirPlay and Companion protocols independently
  • Re-pair individual protocols without disturbing others
  • "Forget & Re-pair All" for a clean credential reset
  • Paired/unpaired status shown per protocol
  • Unpaired devices optionally hidden from main view

Kiosk Configuration

  • Enable/disable kiosk per browser client
  • Landscape or portrait orientation
  • Bind to specific device, room, or "any active"
  • Room binding auto-selects playing over paused
  • Managed remotely from any connected session

Developer Tools

  • Debug panel: live log of commands sent and WebSocket messages received
  • Trigger an immediate mDNS rescan on demand
  • Server restart with two-step confirm (auto-relaunched by launchctl)
  • All settings persisted — survive server restarts
